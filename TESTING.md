# FinderTerm 手動テストチェックリスト

前提: `./scripts/build-app.sh` でビルドし `build/FinderTerm.app` を起動。
アクセシビリティとオートメーション(Finder)の権限を許可済み。

## 基本(仕様: 成功基準)

- [ ] Finderウィンドウを開くと下半分にターミナルが出てプロンプトが表示される
- [ ] Finderウィンドウのドラッグ・リサイズにペインが遅延なく追従する
- [ ] Finderでフォルダ移動するとアイドルなシェルが自動cdされる
- [ ] `claude` (または `sleep 60`) 実行中にフォルダ移動してもcdされない
- [ ] 実行終了後(フォアグラウンドがシェルに戻った後)、Finderの現在フォルダに再同期される
- [ ] Finderウィンドウ2枚 → それぞれ独立したペイン・セッション
- [ ] ⌥⌘Tで全ペインの表示/非表示が切り替わる

## ライフサイクル

- [ ] アイドルなペインのFinderウィンドウを閉じる → ペインが黙って消える
- [ ] `sleep 60` 実行中のウィンドウを閉じる → 確認ダイアログ
- [ ] 「セッションを残す」→ タイトルバー付き独立ウィンドウになりプロセス継続
- [ ] 独立ウィンドウを閉じる(実行中)→ 再度確認ダイアログ
- [ ] 実行中セッションがある状態でアプリ終了(⌘Q)→ 件数付き一括確認
- [ ] `killall Finder` → 実行中セッションは確認ダイアログで「セッションを残す」を選ぶと独立ウィンドウ化されて生き残り、
      Finder復帰後の新ウィンドウには新しいペインが付く

## 追従の網羅

- [ ] 最小化 → ペインが消える。復元 → 戻る
- [ ] 別Spaceへ移動(Mission Control)→ 元のSpaceのペインが他Spaceに出ない
- [ ] Finderをフルスクリーンにするとペインが隠れる
- [ ] ペイン境界のドラッグで高さが変わり、全ウィンドウ・再起動後も維持される
- [ ] 「最近の項目」へ移動 → cdされず、最後の有効フォルダのまま
- [ ] 情報ウィンドウ(⌘I)にはペインが付かない
- [ ] スリープ復帰・外部ディスプレイ抜き差し後も位置がずれない

## エラー系

- [ ] アクセシビリティ権限を剥奪 → 30秒以内にペインが隠れ、メニューバーが警告表示
- [ ] 権限を再許可 → 30秒以内に復帰
- [ ] オートメーション(Finder制御)を拒否 → 位置追従は継続するが自動cdだけ止まる(最後の有効フォルダを保持)
- [ ] ペインで `exit` → ペインが消える(ウィンドウは残る)
- [ ] `claude` をペインで起動して普通に対話できる(矢印キー・色・リサイズ)
